# Description

Ready to challenge your expertise in Music?

Live version at https://masterofmusix.vercel.app/

## How to run
Create a *.env.local* file and set the MUSIXMATCH_API_KEY variable 

```shell
MUSIXMATCH_API_KEY=<YOUR_KEY>
```

### Dev mode
```shell
$ npm run dev
```

### Prod more
```shell
$ npm run build
$ npm run start
```
